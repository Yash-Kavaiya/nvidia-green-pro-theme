# NVIDIA Green Pro Theme v1.0.0

Initial release of a production-level NVIDIA-inspired VS Code theme extension.

## Included

- NVIDIA Green Pro Dark
- NVIDIA Green Pro Light
- 452 workbench colors per theme
- 49 TextMate token rules per theme
- 36 semantic token rules per theme
- Terminal ANSI palette
- Git, diff, merge, notebook, debug, testing, quick input, notification, settings, minimap, and scrollbar colors
- Validated and packaged VSIX asset

## Install from GitHub release

Download `nvidia-green-pro-theme-1.0.0.vsix`, then run:

```bash
code --install-extension nvidia-green-pro-theme-1.0.0.vsix
```
