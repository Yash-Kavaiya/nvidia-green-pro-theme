# NVIDIA Green Pro Theme Palette

## Brand accent

- NVIDIA-inspired green: `#76B900`
- Dark hover/active green: `#4C7F00`
- Light accessible green: `#4C8F00`
- Bright highlight green: `#B8FF5C`

## Dark mode

- Editor background: `#0B0F0C`
- Activity/title background: `#080C08`
- Side/panel background: `#0E140E`
- Foreground: `#E7F2E2`
- Muted foreground: `#9FB29B`
- Borders: `#1F2E18`, `#314327`

## Light mode

- Editor background: `#FCFFF8`
- Activity/sidebar background: `#E8F3E2`, `#F1F7EC`
- Foreground: `#172014`
- Muted foreground: `#5F6E59`
- Borders: `#C8D8BE`, `#B7C9AD`

## Semantic secondary colors

- Cyan/blue: types, interfaces, info, modified Git state.
- Amber/yellow: numbers, constants, warnings.
- Violet: namespaces/modules.
- Red: errors, deletions, breakpoints.
